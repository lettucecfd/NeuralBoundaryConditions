================================================================
Special Requirement: LETTUCE LBM Framework
================================================================

To run the simulations for the Neural-Enhanced Characteristic Boundary Conditions (CBC), as described in the paper (the 2D convected vortex benchmark), the Lettuce LBM framework is required.

**Official Repository:**
https://github.com/lettucecfd/lettuce

Please follow the installation instructions provided in the official Lettuce repository to set up a working environment *before* attempting to run the scripts.
